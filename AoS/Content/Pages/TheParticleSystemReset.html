Reset()

public static void Reset()

The Reset() method readies the Particle System for use, which involves creating
a texture to use when we later draw these particles. This texture is created, and
then all particles are looped and set to inactive. We also set the active count
to 0, which is a value that we will use to count how many particles are active.

After Game1.LoadContent() completes, the game settles into the pattern of
Update(), Draw() being called every 16.6ms. Game passes these Update and Draw
calls onto the Particle System, so let's look at the big Update method now.
We will look at the Spawn() method in it's own article.
