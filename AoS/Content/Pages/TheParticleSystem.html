The Particle System

Where the work gets done

The Particle System class is a Pool written over an Array of Structs.
It has 4 methods, each explained below:

Reset() - sets all particle structs to Inactive.
Spawn() - spawns a particle at X, Y position, with acceleration.
Update() - applies physics, manages particle states, culls, and spawns particles.
Draw() - draws active particles.

Here is the Game loop, which provides some context to the Particle System:
Game1's constructor is called from Program Main, which sets up most of the
common object references, like the reference to the game itself stored in Data.
Then Game1.LoadContent() is called, where the SpriteBatch reference is set,
and we first see Particle System's Reset() method.
