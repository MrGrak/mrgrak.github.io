Draw()

This Draw method opens up SpriteBatch, and starts checking the Particle Array
for active Ids, just like Update() does. When an active Id is found, we copy
the particles position into a Vector2 and pass that into Monogame's SpriteBatch,
along with the particle's color + alpha. We use the texture created before to draw
all the particles with, so all these particles are submitted in one draw call.
